# Test Cases Generated for Requirement 3: Additional Features

**Tổng số:** 5 test cases

## 1. Verify "Remember Me" checkbox functionality
**ID:** TC_ADDITIONAL_001
**Scenario:** User logs in with "Remember Me" checked
**Priority:** Low
**Type:** Functional

**Preconditions:**
User has a valid account.

**Steps:**
1. Navigate to login page.
2. Enter valid Email and Password.
3. Check the "Ghi nhớ đăng nhập" checkbox.
4. Click "Đăng nhập".
5. Close the browser and reopen the application.

**Expected Result:**
User should remain logged in without needing to re-enter credentials.

**Test Data:**
Email: user@example.com, Password: Password123
